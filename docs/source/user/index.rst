User Guide
==========

**ARI Warehouse Management system is designed for companies to manage their warehouses easily. In this system admins of warehouses can add employees, marketplaces, products, cargo companies, providers and supple orders to system. Also admins can edit or delete employees, marketplaces, products, providers and cargo companies. Also admins can see the financial status, stock status and previous orders. Moreover, employees can see orders that attached them. Visitors, employees or admins can create orders.**
